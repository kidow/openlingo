# Instructions

- Do not use TDD in this repository.
- Do not create new tests unless the user explicitly asks for tests.
- Do not modify existing tests unless it is strictly necessary for a user-requested change.
- 각 언어 별로 새로운 지식을 추가할 때는 `src/content/notes/`의 해당 문서를 먼저 확인하고, 필요하면 기존 내용을 재정리한 뒤 반영해라.
- 언어별 노트는 1개 파일만 유지해라.
- `frontmatter`가 있으면 `title`, `description`, `lang`, `createdAt`, `updatedAt`는 유지해라.
- 새로 업데이트할 때마다 `updatedAt` 필드는 유지해라.
